SPA JSON to JSON converter

# SYNOPSIS

**spa-json** *FILE*

# DESCRIPTION

Reads a SPA JSON file, and outputs it as standard JSON.

# EXAMPLES

**spa-json-dump** $(PIPEWIRE_CONFDATADIR)/pipewire.conf
